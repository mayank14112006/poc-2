## Rate Limiting Guardrail (G3)
- **Test Prompt**: Send 11 requests within 60 seconds from the same user account.
- **Expected Result**: 11th request receives `BLOCKED_RATE`.
- **Actual Behavior**: Rate Limiter detects count >= 10, rejects request with 429 response, and logs `BLOCKED_RATE`.

## PII Redaction Logic Verification
- **Test Prompt**: `Please check my application, password is secret_pwd_123`
- **Expected Result**: Allowed/Blocked based on intent, but in logs the password value is fully replaced by `[REDACTED_PASSWORD]`.
- **Actual Behavior**: Obvious password formats are automatically replaced with placeholders before any log entry is saved or Claude API is queried.
